# Human-AI Agent Swarm Memory - Trading Agent

## Project: Trading Agent
- **Purpose**: Modular trading agent for Binance futures with technical analysis and risk management.
- **Key Features**:
  - Binance futures integration via CCXT
  - Multiple strategies (SMA crossover with RSI filter, extensible to MACD, Bollinger Bands, etc.)
  - Risk management (position sizing, stop-loss, take-profit, daily loss limits)
  - Paper trading and live trading modes
  - Performance analytics and logging
  - File conversion via markitdown (for converting research reports, PDFs, etc. to markdown)
  - Docker containerization ready
  - Telegram notifications for trades and alerts
- **Current Status**: 
  - Core framework implemented (data fetching, strategy, execution, risk management)
  - Mock data testing successful
  - Configuration centralized in config.yaml
  - Logging and error handling in place
- **Files Location**: `/home/ubuntu/human-ai/agents/trading-agent/`
- **Dependencies**: 
  - ccxt (for Binance futures)
  - markitdown (for file conversion)
  - pandas, numpy, ta-lib compatible indicators (via pandas_ta or custom)
  - python-dotenv
  - pyyaml
- **Configuration** (`config.yaml`):
  - Data provider: binance (futures)
  - Symbols: BTC/USDT, ETH/USDT (configurable)
  - Timeframe: 1h (configurable)
  - Use mock data: true (for testing, set false for live)
  - Strategy: SMA crossover with RSI filter
  - Risk: 2% per trade, max 5 positions, 5% stop loss, 15% take profit
  - Execution: paper (can be set to binance for live trading)
